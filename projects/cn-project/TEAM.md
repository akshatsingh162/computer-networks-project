## Project Title
Sentinel IDS - Real-time Network Anomaly Detection

## Project Description
A simulation-driven Intrusion Detection System that generates synthetic network traffic data and applies machine learning models (Random Forest, Isolation Forest) to detect anomalies. Includes a web-based dashboard to visualize results in real time.

## Technology Stack
- Python 3.10+
- Flask
- scikit-learn, pandas, matplotlib, seaborn
- Chart.js (for frontend graphs)
- HTML, CSS, JavaScript

## Setup Instructions
1. Clone the repository and enter the team folder inside `projects/`
2. Create a virtual environment:  
   `python -m venv .venv`
3. Activate it:  
   Windows → `.\.venv\Scripts\activate`  
   Linux/Mac → `source .venv/bin/activate`
4. Install dependencies:  
   `pip install -r requirements.txt`
5. Run the app:  
   `python app.py`
6. Open in browser: [http://127.0.0.1:5000](http://127.0.0.1:5000)
